Tribute Page For Our Freedom Fighters

This page contains the case study of Tribute Page for Our Freedom Fighters Open-Source Project which includes the Project Overview, Tools Used and Live Links to the official product.

Project Overview

Tribute Website is an Open-Source project which is a simple and clean multi-page website dedicated to our Freedom Fighters. I created this project for the memorable page of our country's Freedom Fighters to give Tribute to them and i include the information and fast-performing multi-page website without having to code their website from scratch.

This Page is only created by HTML without any CSS and Java Script. It is my first project which i create when i started to learn Web Development.

This Website is now being hosted in Nerlify platform globally and it has helped many students by providing the information about our Freedom Fighters.

Feel free to check out the Project by visiting the Project Link.
